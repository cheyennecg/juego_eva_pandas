## 🧠 Funciones de Python/Pandas que he practicado
A través de este ejercicio, he consolidado el uso de  algunas funciones para el análisis exploratorio de datos:

* **`.describe()`**: Para obtener un resumen estadístico rápido.
* **`.duplicated()`**: Para localizar filas que se repiten y limpiar el dataset.
* **`.dropna()`**: Para eliminar registros que contienen valores vacíos (nulos).
* **`.isnull()`**: Para detectar dónde faltan datos y entender la calidad de la información.
